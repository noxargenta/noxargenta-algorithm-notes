# 常用常数与宏

## INF 哨兵值

```cpp
const int INF = 0x3f3f3f3f;              // ≈ 1e9，够大且不会溢出
const int INF = 0x7f7f7f7f;              // 更大 ≈ 2e9
const long long INFLL = 0x3f3f3f3f3f3f3f3f;  // LL 版本
int maxx = -0x7f7f7f7f;                  // 求最大值初始值
```

- `0x3f3f3f3f` 的优点：两个相加不超 `INT_MAX`，memset 能用
- `0x7f7f7f7f` 更大但两个相加会溢出

## 快读（需要时用）

```cpp
int read() {
    int x = 0, f = 1; char c = getchar();
    while (c < '0' || c > '9') { if (c == '-') f = -1; c = getchar(); }
    while (c >= '0' && c <= '9') { x = x * 10 + (c - '0'); c = getchar(); }
    return x * f;
}
```

## memset vs fill

```cpp
memset(arr, 0x3f, sizeof(arr));      // 初始化为 INF（按字节）
fill(arr, arr + n, 0);               // 任意值
fill(vec.begin(), vec.end(), -1);     // vector
```
