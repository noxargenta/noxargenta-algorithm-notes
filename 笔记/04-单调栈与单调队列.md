# 单调栈与单调队列

## 单调栈 — 右边第一个更大的元素

从 [P_5788](../study/P_5788_模板_单调栈.cpp) 提炼：

```cpp
stack<int> s;
for (int i = n; i >= 1; i--) {
    while (!s.empty() && a[i] >= a[s.top()]) s.pop();
    ans[i] = s.empty() ? 0 : s.top();
    s.push(i);
}
```

### 变体

| 需求 | 遍历方向 | 比较符号 |
|------|---------|---------|
| 右边第一个更大 | 逆序 | `a[i] >= a[s.top()]` |
| 右边第一个更小 | 逆序 | `a[i] <= a[s.top()]` |
| 左边第一个更大 | 正序 | `a[i] >= a[s.top()]` |
| 左边第一个更小 | 正序 | `a[i] <= a[s.top()]` |

## 单调队列 — 滑动窗口最值

从 [P_1886](../study/P_1886_滑动窗口_模板_单调队列.cpp) 提炼：

```cpp
// 窗口最小值
deque<int> q;
for (int i = 0; i < n; i++) {
    if (!q.empty() && q.front() <= i - k) q.pop_front();   // 移出窗口
    while (!q.empty() && a[q.back()] >= a[i]) q.pop_back(); // 维护单调递增
    q.push_back(i);
    if (i >= k - 1) cout << a[q.front()] << " ";
}
```

求最大值只需把 `>=` 改成 `<=`（维护单调递减）。

### 核心思想

- deque 存的是**下标**不是值，方便判断是否在窗口内
- 每次：
  1. 队头出窗口 → pop_front
  2. 队尾破坏单调性 → pop_back
  3. 入队
  4. 窗口满了 → 取队头
