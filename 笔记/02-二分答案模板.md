# 二分答案模板

适用场景：求"最大值最小"或"最小值最大"，且答案有单调性。

## 区间可行性 check — 维护可达范围

从 [D_Jumping_Through_Segments](../competition/20251120ECUT计协第六次训练赛/D_Jumping_Through_Segments.cpp) 提炼：

```cpp
bool check(int k) {
    int L = 0, R = 0;          // 当前可达区间 [L, R]
    for (int i = 1; i <= n; i++) {
        L -= k;  R += k;        // 每步最多走 k，区间扩大
        if (L > r[i] || R < l[i]) return false;   // 和目标区间无交集
        L = max(L, l[i]);        // 取交集
        R = min(R, r[i]);
    }
    return true;
}
```

## 通用二分模板

```cpp
int l = 0, r = INF, ans = -1;
while (l <= r) {
    int mid = (l + r) / 2;
    if (check(mid)) {
        ans = mid;       // mid 可行，尝试更小的（或更大的）
        r = mid - 1;     // 求最小可行解
        // l = mid + 1;  // 求最大可行解
    } else {
        l = mid + 1;
        // r = mid - 1;
    }
}
```

## 注意

- 实数二分用 `for(int i=0;i<100;i++)` 或 `while(r-l>1e-7)`
- 注意 mid 是否会溢出：`mid = l + (r - l) / 2`
