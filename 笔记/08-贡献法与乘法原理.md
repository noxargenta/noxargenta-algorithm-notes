# 贡献法与乘法原理：子串分值和

## 问题

给定一个字符串 `s`，对所有子串，统计每个子串中**不同字符的个数**，求总和。

> 例如 `s = "ABA"`，子串有：`"A"(1) + "AB"(2) + "ABA"(2) + "B"(1) + "BA"(2) + "A"(1) = 9`

## 核心思路：贡献法（Contribution Method）

不枚举所有子串（O(n²) 不可行），而是**计算每个字符在多少个子串中"首次出现"**。

每个字符只在其**第一次出现的位置**贡献计数，这样可以避免重复统计同一个字符的多次出现。

## 乘法原理公式

```cpp
sum += (i - last[s[i] - 'a']) * (s.size() - i);
```

| 部分 | 含义 |
|------|------|
| `i - last[c]` | 左边界可选位置数，从上次出现位置 +1 到当前位置 i |
| `s.size() - i` | 右边界可选位置数，从 i 到字符串末尾 |
| 乘积 | 该字符贡献的子串个数 |

**为什么乘？** 左边界和右边界的选取互相独立，根据乘法原理，总方案数 = 左方案数 × 右方案数。

## 实现细节

- 用 `last[26]` 数组记录每个字符上次出现的位置（初始为 1）
- 字符串前加空格做 1-indexed，方便处理 `i - last[c]`
- 遍历时每处理完一个位置，更新 `last[c] = i`
- O(n) 时间，O(1) 额外空间

## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;
using i64 = long long;
#define endl '\n'
#define int long long

void solve() {
    string s;
    cin >> s;
    s = " " + s;                     // 1-indexed
    int last[26];
    int sum = 0;
    fill(last, last + 26, 0);

    for (int i = 1; i < s.length(); i++) {
        int c = s[i] - 'a';
        sum += (i - last[c]) * (s.size() - i);
        last[c] = i;
    }
    cout << sum << endl;
}

signed main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int _ = 1;
    while (_--) {
        solve();
    }
    return 0;
}
```

## 同类问题

贡献法 + 乘法原理的模式在以下问题中也适用：

- **子串最小值的和**（LeetCode 907）：对每个元素，计算它作为最小值的子串个数
- **子数组范围和**（LeetCode 2104）：分别计算每个元素作为最大值和最小值的贡献
- **统计不同字符数量**的变体题

核心模式：**"每个元素只在其作为某个极值/首次出现时贡献"** + **左右边界独立选择 → 乘法原理**
