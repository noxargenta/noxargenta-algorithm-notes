# 树形 DP：树上括号方案数

## 问题模型

给一棵有根树（根为 1），每个节点赋值为 `(` 或 `)`，要求根到每个叶子的路径构成合法括号串。求方案数模 MOD。

## 思路

关注每个节点处的**括号平衡度**（未匹配的 `(` 数量）。

定义 `dp[u][i]` = 平衡度为 `i` 时，子树 u 内的合法赋值方案数。

这里的"平衡度 i"指：**进入节点 u 之前**还有多少个未匹配的左括号。

## 叶子节点的 DP

叶子到达后必须把括号闭合，所以进入叶子时平衡度必须恰好为 1，叶子赋 `)`：

```
dp[leaf][1] = 1
dp[leaf][i] = 0   (i ≠ 1)
```

## 非叶子节点的转移

考虑节点 u：

- 如果 u 赋 `(`，平衡度变为 `i+1`，所有子节点继承平衡度 `i+1`
  - 子节点方案独立，乘起来：`a = Π dp[v][i+1]`
- 如果 u 赋 `)`，平衡度变为 `i-1`，子节点继承平衡度 `i-1`（要求 `i > 0`，不能关闭不存在的括号）
  - `b = Π dp[v][i-1]`（仅在 `i > 0` 时）

```
dp[u][i] = a + (i > 0 ? b : 0)
```

## 答案

根进入前平衡度为 0，所以答案是 `dp[1][0]`（根只能是 `(`）。

## 代码实现 (O(n²))

```cpp
const int N = 3050;
const int MOD = 998244353;
vector<int> G[N];
int dp[N][N];
int n;

void dfs(int u, int fa) {
    // 叶子
    if (G[u].size() == 1 && fa != 0) {
        dp[u][1] = 1;
        return;
    }

    for (auto v : G[u]) {
        if (v == fa) continue;
        dfs(v, u);
    }

    for (int i = 0; i < n; i++) {
        int a = 1, b = 1;
        for (auto v : G[u]) {
            if (v == fa) continue;
            a = a * dp[v][i + 1] % MOD;
            if (i > 0) {
                b = b * dp[v][i - 1] % MOD;
            }
        }
        if (i == 0) {
            dp[u][i] = a % MOD;
        } else {
            dp[u][i] = (a + b) % MOD;
        }
    }
}
```

## 关键点

| 要点 | 说明 |
|------|------|
| 状态设计 | 以进入节点前的平衡度划分，而不是节点内的平衡度 |
| 叶子特判 | 只有 `dp[leaf][1] = 1`，进入时必须已有 1 个未匹配的 `(` |
| 子节点独立 | 乘法原理，子节点方案互不影响 |
| 根固定 | 根不能是 `)`，所以只加 `a` 不加 `b` |
| 复杂度 | O(n²)，n ≤ 3000 可过 |

## 样例推导

### 链 1-2-3-4

```
dp[4][1] = 1
dp[3][0] = dp[4][1] = 1      → 节点3是'('
dp[3][2] = dp[4][1] = 1      → 节点3是')', i=2→1→0
dp[2][1] = dp[3][2] + dp[3][0] = 2  → 两种: (())、()()
dp[1][0] = dp[2][1] = 2      → 答案 = 2
```

对应两种方案：`(())` 和 `()()`。

### 样例 2 的树

根 1 有三个子树，每个子树独立，最终 `dp[1][0] = 2 × 2 × 1 = 4`。
