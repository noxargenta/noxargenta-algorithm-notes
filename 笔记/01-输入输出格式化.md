# 输入输出格式化

## setfill / setw — 补前导零

```cpp
cout << setfill('0')          // 补什么字符（默认空格）
     << setw(2) << hours << ':' 
     << setw(2) << minutes << ':'
     << setw(2) << ss << endl;
```

- `setw(n)` — 设置下一个输出的**最小宽度**，只对紧跟的一个输出有效
- `setfill(c)` — 宽度不足时填充的字符，一直有效直到被修改
- 头文件：`#include <iomanip>`（但 `bits/stdc++.h` 已包含）

**示例**：`hours=3, minutes=5, ss=9` → 输出 `03:05:09`

### 其他常见格式化

```cpp
cout << fixed << setprecision(2) << x;           // 保留两位小数
cout << hex << x;                                // 十六进制输出
cout << oct << x;                                // 八进制输出
cout << showpos << x;                            // 正数带 + 号
cout << boolalpha << flag;                       // true/false 而不是 1/0
cout << scientific << x;                         // 科学计数法
```
