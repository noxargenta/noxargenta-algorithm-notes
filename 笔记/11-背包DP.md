## 01 背包

每件物品只能选 0 或 1 次。

### 二维 DP

`dp[i][j]` = 前 i 件物品，容量 j 时的最大价值。

```cpp
int T, M;                     // T=容量, M=物品数
int dp[105][1005];
int w[500], v[500];           // w=重量, v=价值

for (int i = 1; i <= M; i++) {
    for (int j = 0; j <= T; j++) {
        if (j >= w[i])
            dp[i][j] = max(dp[i-1][j], dp[i-1][j-w[i]] + v[i]);
        else
            dp[i][j] = dp[i-1][j];
    }
}
cout << dp[M][T] << endl;
```

### 价值变形

开心的金明：价值 = 价格 × 重要度，其余和标准 01 背包完全一样。

```cpp
dp[i][j] = max(dp[i-1][j], dp[i-1][j-v[i]] + v[i] * w[i]);
//                              重量=价格  价值=价格×重要度
```

### 一维优化（倒序）

去掉 i 维度，j 必须**倒序**遍历，保证每件物品只用一次。

```cpp
int dp[1005];
for (int i = 1; i <= M; i++) {
    for (int j = T; j >= w[i]; j--) {
        dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
    }
}
```

## 完全背包

每件物品可以选无限次。

### 一维（正序）

j **正序**遍历，允许同一物品被多次选取。

```cpp
int dp[10000007];
int t, m;                      // t=容量, m=物品数
while (m--) {
    int w, v;
    cin >> w >> v;
    for (int i = w; i <= t; i++) {
        dp[i] = max(dp[i], dp[i - w] + v);
    }
}
cout << dp[t] << endl;
```

边读边算，连数组都不用开。

## 01 背包 vs 完全背包

| | 01 背包 | 完全背包 |
|------|---------|----------|
| 选择次数 | 0 或 1 | 无限 |
| 一维遍历 | `j` 倒序 (`T → w[i]`) | `j` 正序 (`w[i] → T`) |
| 二维递推 | `dp[i-1][j-w[i]]` | `dp[i][j-w[i]]` |
| 代表题 | P_1048 采药 | P_1616 疯狂的采药 |

**关键**：倒序保证每个物品只被用一次（用的是上一行的状态），正序允许同一物品多次使用（用的是当前行刚更新的状态）。
