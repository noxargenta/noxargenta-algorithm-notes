# 差分数组

## 一维差分

区间 `[l, r]` 统一加 `val` 的 O(1) 操作：

```cpp
diff[l] += val;
diff[r + 1] -= val;
```

最后求一遍前缀和还原原数组：

```cpp
for (int i = 1; i <= n; i++) {
    diff[i] += diff[i - 1];
    a[i] = diff[i];    // 或直接输出 diff[i]
}
```

## 二维差分

子矩阵 `(x1,y1)` 到 `(x2,y2)` 加 `val`：

```cpp
diff[x1][y1] += val;
diff[x1][y2 + 1] -= val;
diff[x2 + 1][y1] -= val;
diff[x2 + 1][y2 + 1] += val;
```

还原时做二维前缀和：

```cpp
for (int i = 1; i <= n; i++)
    for (int j = 1; j <= m; j++)
        diff[i][j] += diff[i - 1][j] + diff[i][j - 1] - diff[i - 1][j - 1];
```
