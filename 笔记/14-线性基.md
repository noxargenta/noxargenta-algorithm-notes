# 线性基（Linear Basis / XOR Basis）

来源：[Bitwise_Maximization.cpp](../competition/niukemanyschool2/Bitwise_Maximization.cpp) — 2025牛客多校第二场 B

## 什么是线性基

线性基是 **一组能张成整个 XOR 空间的基向量**。给定一个集合，它的线性基满足：

- 原集合的任意子集 XOR 和，都能由线性基中若干元素的 XOR 表示
- 线性基的任意非空子集 XOR 和 **不为 0**
- 线性基是满足以上条件的最小集合（元素个数 = 向量空间的维数）

通常用一个 `vector<int> xxj(bit, 0)` 数组实现，`xxj[i]` 存储最高位为第 i 位的基向量。

## 插入（构造线性基）

将元素 `x` 插入线性基，从高位到低位尝试：

```cpp
vector<i64> xxj(63, 0);

void insert(i64 x) {
    for (int i = 60; i >= 0; i--) {
        if ((x >> i) & 1) {
            if (xxj[i] == 0) {
                xxj[i] = x;
                break;
            } else {
                x ^= xxj[i];
            }
        }
    }
}
```

**核心思想**：如果当前位没有基向量，就把 x 存进去；否则用已有的基向量消去 x 的这一位，继续往下。

## 查询最大值

在线性基中，求能 XOR 出的最大值：

```cpp
i64 query_max() {
    i64 res = 0;
    for (int i = 60; i >= 0; i--) {
        res = max(res, res ^ xxj[i]);
    }
    return res;
}
```

等价写法（贪心）：从高位到低位，如果异或后变大就异或。

## 查询最小值

线性基中最小的非零元素，就是最小的非零 `xxj[i]`（因为所有基向量的最高位互不相同，最小的那个就是 XOR 能得出的最小非零值）。

## 例题：Bitwise Maximization（最大化两集合 XOR 和）

### 题意

给定数组 `a[1..n]`，把每个数分入两个多重集合，记它们的 XOR 分别为 X 和 Y，求 X + Y 的最大值。

### 推导

1. 记全部数的 XOR 为 S，则 **X ⊕ Y = S**
2. 加法与 XOR 的关系：**X + Y = (X ⊕ Y) + 2(X & Y) = S + 2(X & Y)**
3. 要最大化 X + Y → **最大化 X & Y**
4. X & Y 的某位能为 1 的前提是 S 的这位为 0（若 S 某位为 1，则 X,Y 中恰有一个为 1，AND 得 0）
5. 构造 `mask = ~S`，只保留 S 中为 0 的位
6. 对每个 `a[i] & mask` 建立线性基
7. 从线性基中求出最大 XOR 值 M —— 这就是能得到的最大 X & Y
8. 答案：**ans = S + 2 × M**

### 代码

```cpp
void solve() {
    int n;
    cin >> n;
    int a[n];
    int S = 0;
    for (int i = 0; i < n; i++) {
        cin >> a[i];
        S ^= a[i];
    }

    int mask = ~S;                       // 只保留 S 中为 0 的位
    vector<int> xxj(63, 0);

    for (int i = 0; i < n; i++) {
        int x = a[i] & mask;             // 只考虑可能贡献到 X&Y 的位
        for (int j = 30; j >= 0; j--) {  // a[i] < 2^30
            if ((x >> j) & 1) {
                if (xxj[j] == 0) {
                    xxj[j] = x;
                    break;
                } else {
                    x ^= xxj[j];
                }
            }
        }
    }

    int M = 0;
    for (int i = 30; i >= 0; i--) {
        M = max(M, M ^ xxj[i]);
    }

    int ans = S + 2LL * M;
    cout << ans << endl;
}
```

### 复杂度

- 插入 O(n log A)，查询 O(log A)
- 本题 n 总和 ≤ 5×10⁵，A < 2³⁰

## 线性基经典应用

| 问题 | 做法 |
|------|------|
| 最大 XOR 子集 | 建线性基，query_max() |
| 最小 XOR 子集（非零） | 建线性基，找最小的非零 xxj[i] |
| 第 k 小 XOR 子集 | 线性基高斯消元成最简形式，按位选 |
| 判断一个数能否被表示 | 插入过程中看是否被消成 0 |
| 两集合 XOR 和最大化 | **本题**，S + 2 × query_max_on(a[i] & ~S) |

## 注意事项

- 开 `63` 是基于 64 位整数（`long long`）；若值域 < 2³⁰，开 `30` 够用
- 线性基的数组长度 = 值域的二进制位数，不是 n
- 插入时 `xxj[i]` 存储的是 **最高位为 i 的基向量**，低位可能有其他位的信息
- 线性基可以 **高斯消元化成最简形式**（每行除最高位外只保留一个 1），用于求第 k 小
- 0 无法被线性基表示（xor 空集得 0，但线性基本身不存 0）
